MG1080FY - 印尼海尔项目


早于2017年3月开始的项目合作

初期的两器要求如下：

Overload protector 5TM232RFBYY

Manufacturer Sensata

Rating current 7.4A

Opening temperature 130±5°C

Closing temperature 61±9°C

PTC starting relay QP2-4.7EC

Manufacturer Tianyin

Resistance(25 °C) 4.7±20%Ω

Max. working voltage 180V

Max. working current 12A

Run Capacitor 10μ F/250VAC

经过试验，我是确认能够使用这套两器。
